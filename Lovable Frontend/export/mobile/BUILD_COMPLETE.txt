TROUBLESHOOTING:
================

"App not installed"
→ Free up 500MB storage space
→ Try app-release.apk (universal) instead

"App not compatible"
→ Phone must be Android 5.0 or newer
→ Check: Settings → About Phone → Android version

"Parse error"
→ Re-download the APK (may be corrupted)
→ Try app-release.apk instead

App crashes on startup
→ Close all other apps
→ Restart phone
→ Ensure 1GB+ RAM available

Slow performance
→ Settings → Developer Options
→ Reduce animations to 0.5x
→ Close background apps
